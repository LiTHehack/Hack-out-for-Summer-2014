Tech – advices and stuff

Advices

This hack is about creating the perfect summer app. It doesn't need to be working, a good concept will take you far and a pretty design might win you a price, what we want you to focus on is the idea.

So, before you create the next big awesome cool backend magic thing (Yeah, go you!), take a minute or two and focus on the idea. Make sure you have something to show your friends tonight. Your time at #HOFS2014 is limited, so make the most out of it.

Libraries and tools

Lucky you! The internets is full of great tools and libraries to make your life easier in situations such as these. Just make sure that you don't get stuck at something just because you wanted try something new!

Some of our favourite tools and libraries are:

  • AngularJS JS
  • Bootstrap CSS
  • CodeIgniter PHP
  • Less CSS Dev
  • jQuery JS
  • Sencha Touch (there's a special open source license) JS
  • normalize.css CSS
  • gmaps.js JS
  • SimpleGMaps JS
  • Ionic JS CSS
